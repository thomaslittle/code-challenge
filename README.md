# code-challenge
Ally Code Challenge

Installation
-----------
Clone this repo

Add node modules
```
npm install
```

Add Bower components
```
bower install
```

Run it!
```
gulp serve
```

Build it!
```
gulp serve
```

Deploy to surge (http://thomaslittle-ally.surge.sh/)
```
gulp deploy
```

[View demo](http://thomaslittle-ally.surge.sh/)
